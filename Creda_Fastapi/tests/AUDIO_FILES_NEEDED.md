## How to Prepare Audio Files

### Option 1: Using Your Phone
1. Use a voice recording app (WhatsApp Voice, Voice Memos, etc.)
2. Read the text aloud clearly
3. Save as WAV format (convert if needed)
4. Place in `tests/audio/` folder

### Option 2: Using Python (Recommended)
```python
# Install text-to-speech library
pip install pyttsx3 soundfile

# Generate audio from text
import pyttsx3
import soundfile as sf

engine = pyttsx3.init()
engine.setProperty('rate', 150)  # Natural speech rate
engine.save_to_file("What is SIP?", "tests/audio/en_sip_question.wav")
engine.runAndWait()
```

### Option 3: Using Online TTS Services
- Google Cloud Text-to-Speech
- Azure Speech Services
- Microsoft Cortana
- Export as WAV, save to `tests/audio/` folder

### Option 4: Use Existing Audio
If you have other voice files (recordings, etc.), convert to WAV:
```bash
# Using ffmpeg
ffmpeg -i input_file.mp3 -acodec pcm_s16le -ar 16000 -ac 1 output.wav
```

## Troubleshooting

### "File not found" Error
```
ERROR: Could not find tests/audio/en_sip_question.wav
```
**Solution:** Check that files are in `tests/audio/` folder (not `tests/` or another location)

### WAV Format Issues
```
ERROR: wav file invalid or corrupted
```
**Solution:** Ensure files are 16-bit PCM, 16kHz mono WAV format

### Audio Too Long/Short
```
WARNING: Audio duration not in expected range
```
**Solution:** Files should be 2-5 seconds. Longer files will be truncated or cause timeout.

### Test Skipped Due to Missing Audio
```
SKIPPED: Audio file not found (test/audio/file.wav)
```
**Solution:** Tests will auto-skip if audio files don't exist (won't fail). It's safe!
